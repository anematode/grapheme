Class: RealInterval

RealInterval(min, max, defMin, defMax, contMin, contMax)

new RealInterval(min, max, defMin, defMax, contMin, contMax)

Construct a RealInterval. Only a single argument is needed, and will produce an "exact interval", but there are six arguments in total. The meaning of each argument is described in the class description.
Parameters:
Name Type Default Description
min number The minimum of the interval.
max number The maximum of the interval.
defMin boolean true Whether the interval is always defined.
defMax boolean true Whether the interval is potentially defined.
contMin boolean true Whether the interval is always continuous.
contMax boolean true Whether the interval is potentially continuous.
